# PlantUML Diagram Generator

Claude Code skill for generating PlantUML diagrams.

## References

All syntax references are sourced from https://plantuml.com/. Copyright belongs to PlantUML.

| Type | Description |
|------|-------------|
| Class | 类图 - 类结构和继承关系 |
| Sequence | 序列图 - 消息交互流程 |
| Activity | 活动图 - 工作流/业务流程 |
| State | 状态图 - 对象状态转换 |
| Component | 组件图 - 系统组件结构 |
| Deployment | 部署图 - 软硬件部署拓扑 |
| Use Case | 用例图 - 用户与系统交互 |
| Object | 对象图 - 对象实例快照 |
| Timing | 时序图 - 基于时间的状态变化 |
